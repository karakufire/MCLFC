* mclfc - Minecraft Language File Comparator  *
* * * * * * * * * * * * * * * * * * * * * * * *

Usage: mclfc [options] <original> <translated>

Options:
  -V, --version        output the version number
  -o, --output <PATH>  set output file path
  -d, --differences    outputs different lines, untranslated entries and no-in-original entries
  -u, --untranslated   outputs untranslated entries only
  -h, --help           output usage information
